      We grabbed the same screenshot for each of the versions to provide some comparison material.

      1. The setup start screen
      2. The login screen
      3. The dashboard with default widgets
      4. The package browser home screen
      5. Package management with TinyMCE installed
      6. Editing the home resource with focus on the pagetitle
      7. Browsing an assets/uploads/ folder with 4 uploaded images, the first selected, using the media browser
      8. The system settings page
      9. Editing a user group.

      We repeated this cycle for MODX Revolution 2.0.8, 2.1.5, 2.2.15 and 2.3.3 - the most recent version in each of the minor releases to date.
