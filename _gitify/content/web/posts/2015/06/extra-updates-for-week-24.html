              • Google Analytics Dashboard Widget 1.0.4-pl, released on 2015-06-08 View Changelog

                ×

                Analytics Dashboard widget 1.0.4

                ====================================

                -Fixed errors in modx log #5

                Analytics Dashboard widget 1.0.3

                ====================================

                - Updated Google OAuth

                - Updated all API requests

                - Changed fix for MODX 2.3 layout

                - Fixed cache when changing profile

                Analytics Dashboard widget 1.0.2

                ====================================

                - Added quick fix for the dashboard widget layout

                Analytics Dashboard widget 1.0.1

                ====================================

                - All data-charts are working again

                - Changed MODX.on('ready' to Ext.ready(

                - Authentication changed to make domain-verification fool-proof

                - Updated the Google's API for listing all the accounts

                - Added loading-screen after Analytics profile is selected

                Analytics Dashboard widget 1.0.0

                ====================================

                - Vertical tabs

                - Added Internel search

                - Ability to hide tabs

              • Updater 0.2.10-beta, released on 2015-06-08 View Changelog

                ×

                0.2.7-beta -> 0.2.10-beta

                • changed notifier event to OnWebPageComplete
                • fixed critical bug when github api only returns unusable message
                • fixed widget warning when on dev-branch/github
                • adjusted colors
                • list every package in notifications in one line

              • SimpleCart 1.3.0-dev1, released on 2015-06-09 View Changelog

                ×
                • SimpleCart is now maintained and distributed by modmore.com
                • Include all translations in the main SimpleCart package instead of separate language packs.
                • Make sure cookies set by SimpleCart adhere to session_cookie_path and session_cookie_domain settings

              • ContentBlocks 1.2.6-pl, released on 2015-06-09 View Changelog

                ×
                • Fix issue where removing layouts did not affect the layouts-per-page calculation
                • Fix ultimate parent-based default template selection (#291)
                • Fix TinyRTE missing styling and buttons bar on Windows/Firefox (#287)
                • Add some missing indices to the database tables (#290)
                • Make sure select option recognises an empty string as value (#288)
                • Fix upload to media sources with a full url as baseUrl
                • Set Ace minLines option to 1 for smaller code inputs
